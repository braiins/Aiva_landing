Why this space

The self-monitoring space is very fragmented and there is yet to be a unifying platform gathering actionable, real-time data. For the majority of patients, the only alternative is manual data input by caregivers or health professionals. This results in a scattered, incoherent, dated, and largely inactionable dataset for everyone involved. We are creating something which is real-time, is not limited by a single type of data input and provides actionable output through predictive analytics.

How it works

Our product consists of a set of ambient sensors with extremely long battery life, a base station and Aiva emergency response software. Sensors are placed into a living environment of the cared person.

The Aiva system processes and learns from events and issues alerts to operators when any emergency situation is detected or when e.g. a disease symptom is found.
